My Blog

The Challenge to Create the Project in Higher Education Innovation Awards 2024

July 10, 2025

Based on the creation of Project The Game for people who have Alzheimer’s disease or are at risk of getting the disease by User-Generated Content for the competition, our original goal was to create an application with Flutter and Unity, where Flutter is used for app development and Unity is utilized for game development.

Unfortunately, in the few week before the competition, we had to switch to Flutter development because Unity only allows for static apps, which did not fit the idea of our app.

Despite the fact that many things did not go as planned, we were able to fully construct this application. Developing this project showed me that we should thoroughly test the functions of the program before using it.
